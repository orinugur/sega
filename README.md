# 세이크리드 가드 데미지 계산기 (Sacred Guard Damage Calculator)

마비노기 아르카나 재능인 **세이크리드 가드(Sacred Guard)**의 전용 액티브 스킬 및 베이스 재능 스킬들의 데미지를 계산하고 스펙별 빌드를 비교해 볼 수 있는 프리미엄 시뮬레이터 웹 애플리케이션입니다.

---

## 🌟 주요 기능

1. **실시간 데미지 시뮬레이션**:
   - 맥댐, 방어력, 생명력 등 스탯 입력을 기반으로 세이크리드 가드 8종 아르카나 스킬 및 4종 재능 스킬의 일반/크리티컬 데미지를 즉시 계산합니다.

2. **카드 내 수식 분해 연동**:
   - 데미지 결과 카드를 클릭하면 하단으로 카드가 확장되면서 **적용된 버프, 계산에 사용된 계수, 방보에 의한 감쇄율** 등 1단계부터 4단계까지의 연산 과정이 완벽히 시각화되어 표시됩니다.

3. **로컬 스토리지 빌드 저장소**:
   - 현재 입력 세팅에 이름을 부여하여 로컬 브라우저에 누적으로 저장하고 보관할 수 있습니다.
   - **불러오기**: 저장했던 모든 세팅과 입력값을 복원합니다.
   - **비교기준**: 특정 빌드를 기준 빌드로 설정하면, 다른 세팅으로 조율 시 데미지의 실시간 증감율 배지(`+X.X%` / `-Y.Y%`)가 결과 카드에 표시됩니다.

4. **최신 밸런스 공식 반영**:
   - 에르그 및 어둠의 에르그 연산 타이밍, 성찰의 흔적 버프 배율(유물 레벨 연동), 급소관통 디버프(방어 비율 감소), 곱연산 디버프(모모, 데스마커) 등 최신 연구 수식을 모두 반영했습니다.

---

## 📂 파일 구조

- `index.html`: 반응형 및 다크 테마 기반의 레이아웃 HTML
- `css/style.css`: 테마 스타일, 폰트, 호버 애니메이션, 반응형 레이아웃 스타일시트
- `js/data.js`: 몬스터(방보/피저), 무기(스매시/배쉬 보너스), 방패 데이터셋
- `js/calculator.js`: 연산 핵심 로직 및 이벤트 바인더

---

## 🚀 실행 및 배포 방법

이 프로젝트는 별도의 서버 구성이나 프레임워크 빌드 과정이 필요 없는 순수 **Client-Side (Vanilla JS)** 애플리케이션입니다.

1. **로컬에서 실행**:
   - 프로젝트 폴더의 `index.html` 파일을 크롬(Chrome)이나 에지(Edge) 등 웹 브라우저에 끌어다 놓기(Drag & Drop)하거나 더블 클릭하여 즉시 실행할 수 있습니다.

2. **웹에 배포 (GitHub Pages 등)**:
   - 파일 구조 그대로 GitHub 저장소에 올린 후, 저장소의 **Settings > Pages** 탭에서 배포 소스를 `main` 혹은 `master` 브라우저로 선택하여 저장하면 몇 초 안에 무료 웹 호스팅 주소로 배포됩니다.
